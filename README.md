## ⚙️ Contact Form

The contact form is powered by **EmailJS** — no backend required. Form submissions are delivered directly to `info@skillbrig.co.in`.

To configure, update these 3 values inside `index.html`:

```js
const EJS_PUBLIC_KEY  = "YOUR_PUBLIC_KEY";
const EJS_SERVICE_ID  = "YOUR_SERVICE_ID";
const EJS_TEMPLATE_ID = "YOUR_TEMPLATE_ID";
```
